# PVSioParser

This library provides the following utilities for writing simple parsers in PVSio.

* [`pvsio_regex`](./pvsio_regex.pvs): An implementation of regular expressions
* [`pvsio_lex`](./pvsio_lex.pvs): A simple lexer
* [`pvsio_tokenizer`](./pvsio_tokenizer.pvs): A simple tokenizer
